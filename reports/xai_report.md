# Explainable AI Report

## Tools Used

- SHAP
- LIME

## Global Insights

The most influential features identified by SHAP are:

- External credit scores
- Debt-to-income ratio
- Loan burden ratio
- Employment duration
- Credit utilization indicators

## Local Insights

Individual predictions can be explained using:

- SHAP waterfall plots
- LIME local explanations

These techniques reveal how specific features influence each customer's predicted default risk.

## Business Value

Explainable AI improves:

- Transparency
- User trust
- Regulatory compliance
- Fair lending practices

## Recommendation Engine

A rule-based recommendation engine generates personalized suggestions to help customers improve their creditworthiness.

Example recommendations include:

- Reduce debt relative to income
- Avoid multiple credit applications
- Maintain stable employment
- Improve repayment behavior
